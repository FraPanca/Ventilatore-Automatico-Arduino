# Ventilatore-Automatico-Arduino
Un ventilatore che si attiva quando si supera una certa soglia di temperatura.

Realizzazione di un ventilatore con una scheda Arduino UNO R3, con accensione automatica quando la temperatura esterna Ã¨ superiore ad una certa soglia.

Componenti necessari:
  - UNO R3 Controller Board
  - Fan Blade
  - 3-6V Motor CC
  - Button
  - LCD1602 Module (with pin header)
  - Potentiometer 10k
  - NPN Transistor PN2222
  - Diode Rectifier
  - Thermistor
  - 3 resistor 220 Ohm
  - 1 resistor 10 kOhm
  - Breadboard
  - Breadboard jumpers
