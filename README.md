# Ventilatore-Automatico-Arduino

## Italiano
Un ventilatore che si attiva quando si supera una certa soglia di temperatura.

Realizzazione di un ventilatore con una scheda Arduino UNO R3, con accensione automatica quando la temperatura esterna è superiore ad una certa soglia.
Tramite bottone può essere disattivata l'accensione automatica.
Il display LCD mostra delle informazioni sulla temperatura.

### Componenti necessari
- UNO R3 Controller Board
- Fan Blade
- 3-6V Motor CC
- Button
- LCD1602 Module (with pin header)
- Potenziometro 10k
- NPN Transistor PN2222
- Diode Rectifier
- Thermistor
- 3 resistori 220 Ohm
- 1 resistore 10 kOhm
- Breadboard
- Breadboard jumpers

## English
A fan that activates when a certain temperature threshold is exceeded.

Implementation of a fan using an Arduino UNO R3 board, which turns on automatically when the external temperature exceeds a certain threshold.
The automatic activation can be disabled via a button.
The LCD display shows information about the temperature.

### Required components
- UNO R3 Controller Board
- Fan Blade
- 3-6V Motor CC
- Button
- LCD1602 Module (with pin header)
- 10k Potentiometer
- NPN Transistor PN2222
- Diode Rectifier
- Thermistor
- 3 resistors 220 Ohm
- 1 resistor 10 kOhm
- Breadboard
- Breadboard jumpers
